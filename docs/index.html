The binsight.ai App :herb:

About/Overview

A cross-platform mobile app built with Flutter/Dart for users of Soundbendor Lab's Smart Compost Bin. The app enables Wi-Fi connectivity for the bin, so participants in a field study will be able to use their bin to track household food waste and follow data trends over time.
