List<RouteBase> routes
getter/setter pair

The routes for the application.

The routes are defined like a tree. There are two top-level routes: 'main' and 'set-up'. The 'main' route is wrapped in a ShellRoute to share the bottom navigation bar. The ShellRoute returns an NavigationShell widget, which contains the top navigation bar.
